I-9 Employment Eligibility

Ensure the employee presents valid documents for I-9 verification.

What ID does this employee need to bring?
  • Option A (List A): US Passport/Passport Card, Green Card, Employment Authorization Card, Foreign Passport + I-551 stamp
OR
  • Option B (List B + List C):
  • List B = Driver's license, State ID, School ID w/photo;
  • List C = Social Security card, Birth certificate, US Citizen ID Card
SSN is always required for payroll — regardless of which ID combo you collect. If they don't have the card, get the number verbally.
I-9 Documentation

List B + C — two documents
