I-9 Employment Eligibility

Select which documents the employee presented, enter the details, and upload photos or scans of each.

What ID does this employee need to bring?
They have two options — either works on its own.
Option A — one document from List A
  • U.S. Passport or Passport Card
  • Permanent Resident Card (Green Card / I-551)
  • Employment Authorization Document (I-766)
  • Foreign Passport + I-551 stamp or I-94
— OR —
Option B — one from List B + one from List C
  • List B (proves identity): Driver's license, State ID, School ID with photo, U.S. Military ID card or draft record, Military dependent's ID card, Native American tribal document
  • List C (proves work authorization): Social Security card (unrestricted), U.S. Birth Certificate, Certification of Birth Abroad (FS-240 or DS-1350), U.S. Citizen ID Card (I-197), Native American tribal document
⚠️ SSN is always required for payroll — regardless of which ID combo you collect. If they don't have the card, get the number verbally and record it on the Comp Plan.
